# ChatFlow – Real-Time Messaging App for iOS

**ChatFlow** is a modern real-time messaging application for iOS, built with **SwiftUI** and **Firebase** using the **MVVM (Model-View-ViewModel)** architecture. The app delivers a fast, secure, and responsive chat experience with real-time synchronization, clean UI, and scalable architecture.

## ✨ Features

* 🔐 Firebase Authentication (Sign Up & Sign In)
* 💬 One-to-One Real-Time Messaging
* ⚡ Live Message Synchronization with Cloud Firestore
* ✅ Message Status (Single Tick & Double Tick)
* 🗑️ Delete for Me & Delete for Everyone
* 📋 Copy Messages
* 📤 Share Messages
* 👤 User Profiles
* 🖼️ Profile Image Support
* 🔍 User Search
* 📱 Modern SwiftUI Interface
* 🏗️ MVVM Architecture
* ☁️ Cloud Firestore Database
* 🔄 Real-Time Data Updates

## 🛠️ Tech Stack
