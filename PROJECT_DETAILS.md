ChatFlow is a modern iOS real-time messaging application developed using SwiftUI and Firebase following the MVVM architecture. It provides a smooth and secure chatting experience with real-time message synchronization and a clean, user-friendly interface.

Features
Firebase Authentication (Sign In / Sign Up)
One-to-One Real-Time Messaging
Live Message Updates using Firestore
User Profiles
Online User Status (if implemented)
Profile Image Support
User Search
Modern SwiftUI Interface
MVVM Architecture
Firebase Firestore Database

Swift
SwiftUI
MVVM
